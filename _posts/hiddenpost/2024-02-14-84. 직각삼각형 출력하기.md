```swift
import Foundation

let n = readLine()!.components(separatedBy: [" "]).map { Int($0)! }

var string = "*"

for i in 0..<n[0]{
    print(string)
    string += "*"
}
```
---
처음에 n을 그대로 넣으려고했는데 타입에러가나서 자세히보니 배열이어서
어차피 하나의 수만 입력을 받기에 n[0]으로 하였다.

그리고 *가 한개씩 늘어나게 하였다.

다른코드 보니 for each와 map joined을 사용하여 간결하게 하였다.

### 다른코드
```swift
import Foundation

let n = readLine()!.components(separatedBy: [" "]).map { Int($0)! }

(1...n[0]).forEach {
    print((1...$0).map { _ in "*" }.joined())
}
```
