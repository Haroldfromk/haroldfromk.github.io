```swift
import Foundation

func solution(_ num_list:[Int], _ n:Int) -> [Int] {
    
    var answer : [Int] = []

    answer = Array(num_list[n...]) + Array(num_list[..<n])

    return answer
}
```

---
이번에도 슬라이싱을 통해서 문제를 해결하려고 하였다.

n번쨰 까지에서 내가 잘못 이해했는지 ... 으로 했을때는 하나를 더 포함을 해서
..< 으로 바꾸어 해결하였다.
