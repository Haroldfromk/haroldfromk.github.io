```swift
func solution(_ s:String) -> String {
    
    var string : String = ""
    
    string = s.map{String($0)}.sorted(by:>).joined()

    return string
}
```
---
배열로 변환하여 내림차순으로 한뒤 다시 join으로 하면서 문자열을 하나로 맞추었다.
