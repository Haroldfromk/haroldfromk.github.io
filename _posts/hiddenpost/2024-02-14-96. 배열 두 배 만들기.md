```swift
import Foundation

func solution(_ numbers:[Int]) -> [Int] {
    
    var arr : [Int] = []
    
    arr = numbers.map{$0*2}
    
    return arr
}


```
---
반복문 대신 map을 사용하였다. 

$0으로 배열 각각의 인자에 2를 곱한 값을 리턴하여 답을 구하였다.
