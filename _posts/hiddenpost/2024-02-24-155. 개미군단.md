```swift
import Foundation

func solution(_ hp:Int) -> Int {
    
    var answer = 0
    
    answer = hp/5 + (hp%5)/3 + ((hp%5)%3)/1
    
    return answer
}
```
---
단순하게 생각을 했다.
먼저 hp에서 장군개미가 얼마나 필요한지를 구하기위해 hp/5를 한다.

그리고 남은 나머지에서 3을 나누고

그나머지에 1을 나누었다. (사실 마지막은 필요없긴한데... 깔맞춤했다)

이렇게해서 합을 리턴하였다.
