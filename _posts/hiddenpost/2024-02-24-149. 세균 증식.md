```swift
import Foundation

func solution(_ n:Int, _ t:Int) -> Int {
    
    var answer : Int = 0
    
    answer = n * Int(pow(Double(2),Double(t)))
    

    return answer
}
```
---

2의 t제곱씩 수가 증가하는것을 보고 pow를 사용하여 결과 값을 리턴하였다.

pow는 안에 들어가는 매개변수가 소수점의 형태이어야 하므로 Float or Double Type으로 해주어야한다.
