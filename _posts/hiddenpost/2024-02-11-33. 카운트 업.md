나의 코드
```swift
import Foundation

func solution(_ start_num:Int, _ end_num:Int) -> [Int] {
    
    var answer : [Int] = []
    
    for i in start_num ... end_num {
        answer.append(i)
    }
    
    return answer
}
```

이렇게 생각한 이유.

반복문을 통해 start~end로 범위를 잡고 그때의 i값을 append를 통해 추가하였다.
