나의 코드 (24.1.19)
```swift
import Foundation

var answer = 0

func solution(_ num1:Int, _ num2:Int) -> Int {
    
    if 0 < num1 && num1 <= 100 {
        if 0 < num2 && num2 <= 100 {
            answer = num1 / num2
            
        }
    }
    
    return answer
}
```
이렇게 생각한 이유.

제한사항을 반영하여 코드를 작성하기 시작하였고, 몫을 구하는 것이기 때문에 단순 연산자인 / 을 사용하여 몫을 구했다.

해당코드는 추후에 더 공부하여 개선점을 찾아 수정할 예정이다.
