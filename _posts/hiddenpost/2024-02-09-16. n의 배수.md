나의 코드
```swift
import Foundation

func solution(_ num:Int, _ n:Int) -> Int {
    
    var answer = 0
    
    answer = num % n == 0 ? 1 : 0
    
    return answer
}
```

이렇게 생각한 이유.

n의 배수이므로 num을 n으로 나눴을때 무조건 나머지가 0이 되어야 한다.

그래서 삼항연산자를 통해 간략하게 표현을 했다.
