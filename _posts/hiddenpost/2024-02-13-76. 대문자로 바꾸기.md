```swift
import Foundation

func solution(_ myString:String) -> String {
    
    var answer : String = ""
    
    answer = myString.uppercased()
    
    return answer
}


```
---
문자열을 대문자로 바꿔주는 uppercased를 사용하여 해결하였다.
