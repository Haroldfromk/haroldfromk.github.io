```swift
import Foundation

func solution(_ n:Int, _ k:Int) -> Int {
    
    var answer : Int = 0
    
    answer = n * 12000 + (k-(n/10)) * 2000
    
    
    return answer
}
```
---
10인분에 1개씩이므로 서비스는 n/10 의 몫으로 받는다는 것을 알고
음료수의 갯수 - n/10 으로 계산하여 답을 구했다.
