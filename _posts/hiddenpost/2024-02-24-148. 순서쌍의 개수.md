```swift
import Foundation

func solution(_ n:Int) -> Int {
    
    var answer : Int = 0
    
    for i in 1...n {
        if n%i == 0 {
            answer += 1
        }
    }
    
    return answer
}
```
---
순서쌍으로 포장했지만 자세히보면 약수의 개수를 묻는 문제였다.

자연스럽게 for문을 써서 풀었는데, 저번에 비슷한문제를(1...n).filter~ 

이런식으로도 풀려고했던기억이 있는데 너무나도 당연하게 for문을 써버렸다.
