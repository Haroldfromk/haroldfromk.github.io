```swift
import Foundation

func solution(_ start:Int, _ end_num:Int) -> [Int] {
    
    var answer : [Int] = []  
    
    answer = (end_num ... start).map{$0}

    return answer.sorted(by:>)
}
```

이렇게 한 이유

map을 사용하여 해결하려고 하였고 end~start까지의 범위로 내림차순 정렬로 리턴하여 해결하였다.
