TestFlight 심사를 기다리는 동안 새로운 미니 프로젝트를 하나 해보려고 한다.

프로젝트 이름은 JustChat. 소셜 로그인과 Firebase를 이용한 실시간 채팅 기능을 구현하는 앱이다.

이 주제를 고른 이유는 2년 전 분식집을 주제로 한 팀 프로젝트에서 이 부분(소셜 로그인, 채팅)을 구현할 때의 아쉬움 때문이다. 

당시엔 관련 지식과 경험이 부족해서 AI에게 구현을 거의 맡기다시피 했고, 일정도 빠듯해서 그 과정을 제대로 정리하지도 못했다. 그때 못 다한 이해와 정리를, 이번엔 시간에 쫓기지 않는 상태에서 제대로 해보고 싶다.

일단 UI의 경우 RunWay와 똑같이 AI에게 만들어 달라고 했다.

우선 첫화면은 다음과 같다.

![](https://pub-1fd8ca6711bd4f3f8b74d88a697b50f9.r2.dev/2026-07-06-JustChat1/homeview.png){: width="50%" height="50%"}

다만 소셜로그인 기능을 구현하기전에 먼저 전체적인 진행 순서를 간단하게 잡고 가려한다.

1. **Firebase 연동** - GoogleService-Info.plist 설정, SDK 연결, Auth/Firestore 콘솔 세팅
2. **소셜 로그인 연동** - Apple/Google Sign-In, async/await로 감싸기
3. **채팅 기능 구현** - Firestore 실시간 리스너, 메시지 송수신

모델링은 각 단계 진행하면서 필요할 때마다 그때그때 확정해나갈 생각이다. 처음부터 스키마를 다 잡고 시작하기보다, 실제로 붙여보면서 필요한 형태로 다듬는 쪽이 낫다고 판단했다.

## Firebase 연동

오래간만에 Firebase를 쓰는 것 같다.
그때는 UIKit 기반이었는데, 이번엔 SwiftUI + async/await로 새로 짜는 거라 접근 방식이 다르다.

### 프로젝트 생성 및 앱 등록

Firebase Console에서 프로젝트를 만들고 iOS 앱을 등록한다.

등록하면 `GoogleService-Info.plist`를 다운로드하라고 안내가 나온다.

### SDK 추가 (SPM)

필요한 것만 추가한다. `FirebaseCore`는 나머지 세 개가 내부 의존성으로 갖고 있어서 따로 추가하지 않아도 된다.

1. FirebaseAuth
2. FirebaseFirestore
3. FirebaseStorage (사진 첨부 기능 때 사용 예정)

패키지 추가 시 **target을 반드시 확인**할 것. None으로 설정하면 import가 안 된다.

## FirebaseApp 초기화

Firebase SDK를 추가했으면 앱 진입점에서 초기화를 해줘야 한다.

[Firebase Docs](https://firebase.google.com/docs/ios/setup?hl=ko){:target="_blank"}에서는 SwiftUI 앱이라도 `AppDelegate`를 만들어서 초기화하는 방식을 권장한다.

Firebase 내부적으로 `UIApplicationDelegate`의 콜백(푸시 알림, URL 스킴 처리 등)에 의존하는 부분이 있기 때문이다. 

`@UIApplicationDelegateAdaptor`는 SwiftUI 앱에서 `AppDelegate`를 다시 연결해주는 브릿지 역할을 한다.

```swift
class AppDelegate: NSObject, UIApplicationDelegate {
    func application(_ application: UIApplication,
                     didFinishLaunchingWithOptions launchOptions: [UIApplication.LaunchOptionsKey : Any]? = nil) -> Bool {
        FirebaseApp.configure()
        return true
    }
}

@main
struct JustChatApp: App {
    
    @UIApplicationDelegateAdaptor(AppDelegate.self) var delegate

    var body: some Scene {
        WindowGroup {
            RootView()
        }
    }
}
```

이렇게 기본작업은 끝났다.
