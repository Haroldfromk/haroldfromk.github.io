3. firebase SDK 추가하기

시간이 지나면서 여러 방식으로 추가를 할 수 있게 되었다.

여기선 cocoapods으로 설치를 진행해볼것이다.

// podfile
platform :ios, '13.0'

target 'Flash Chat iOS13' do
  use_frameworks!

  # Pods for Flash Chat iOS13

  pod 'FirebaseAuth'
  pod 'FirebaseFirestore'

end
