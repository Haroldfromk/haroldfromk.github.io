import Foundation

func solution(_ arr:[String]) -> String {
    
    var answer : String = ""
    
    for i in arr {
        answer += i
    }
    
    return answer
}

이렇게 생각한 이유.

반복문을 돌려서 answer라는 변수에 누적하였다.

정수는 계속 더해지지만 문자는 이어서 붙이게 된다.
